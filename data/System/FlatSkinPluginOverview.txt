---++ Twistys
To get the default twisty use the default twisty icon (=%<nop>TMPL:P{"TwistyShowIcon"}%=) in the link parameter:

To style another icon as twisty icon, use the CSS-Class ="ma-twisty-icon"=.
<verbatim class="tml">
%TWISTY{
    showlink="%TMPL:P{"TwistyShowIcon"}% %MAKETEXT{"Show form..."}%"
    hidelink="%TMPL:P{"TwistyHideIcon"}% %MAKETEXT{"Hide form..."}%"
}%
%ENDTWISTY%
</verbatim>
%TWISTY{
    showlink="%TMPL:P{"TwistyShowIcon"}% %MAKETEXT{"Show form..."}%"
    hidelink="%TMPL:P{"TwistyHideIcon"}% %MAKETEXT{"Hide form..."}%"
    remember="off"
    start=hide"
}%

---+++ Normal Tabs
To get normal Tabs you can use the Tabforeach Macro:

<verbatim class="tml">
%TABFOREACH{
  "New, Managed, Delete"
  titleformat="$title"
  urlformat="%SCRIPTURL{rest}%/RenderPlugin/template?topic=%WEB%.WebHomeViewTemplate;name=WebHomeView;expand=%22getRiskData%22%20status%3D%22$value%22%20pdf%3D%22off%22;"
}%
</verbatim>
%TABFOREACH{
  "New, Managed, Delete"
  titleformat="$title"
  urlformat="%SCRIPTURL{rest}%/RenderPlugin/template?topic=%WEB%.WebHomeViewTemplate;name=WebHomeView;expand=%22getRiskData%22%20status%3D%22$value%22%20pdf%3D%22off%22;"
}%
